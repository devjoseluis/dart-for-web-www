Operators in Dart are similar to JavaScript.

### Arithmitic

| Operater | JS Equivelent |Description |
|----------|---------------|------------|
| +        | +             | Addition or Concatenation |
| -        | -             | Subtraction |
| *        | *             | Multiplication |
| /        | /             | Division |
| %        | %             | Modulo |
| ++       | ++            | Increment |
| --       | --            | Decrement |

### Booleans

| Operater | JS Equivelent | Description |
|----------|---------------|------------|
| !        | !             | Not operator |
| ==        | ==           | equal    |
| !=       | !=            | Not equal |
| <        | <             | less than    |
| <=       | <=            | less than or equal to |
| >        | >              | greater than    |
| >=       | >=            | greater than or equal to |

### Etc

| Operater | JS Equivelent | Description |
|----------|---------------|------------|
| //       | //            | comment    |
